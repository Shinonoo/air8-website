Put your product photos here (jpg or png, roughly 800x800 or 4:3).

Name each file to match the "image" path in products-data.js, e.g.:
  elta-ap-series.jpg
  elta-centrifugal.jpg

If an image file is missing, the card/modal shows a clean fallback
tile instead of a broken image, so the site never looks broken while
you're still gathering photos.

Tip: compress photos (e.g. squoosh.app) so pages stay fast.
