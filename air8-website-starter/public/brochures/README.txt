Put your product brochure PDFs in this folder.

Name them to match the "brochure" paths in products-data.js, e.g.:
  tiger-wvs.pdf
  abie-duct-silencer.pdf
  elta-centrifugal.pdf

Only the PDFs a visitor actually clicks get downloaded, so having
many brochures here does NOT slow the website down.

Tip: keep each PDF reasonably small (compress if over ~5 MB).
